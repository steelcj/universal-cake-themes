# Stijl Core

**Stijl Core** is a responsive, WCAG AAA–compliant web theme inspired by the De Stijl design movement.

It combines geometric clarity, bold primary color blocks, and modern accessibility standards into a lightweight, dependency-free foundation for structured websites.

## Purpose

Stijl Core serves as a clean, accessible base theme for:

- Documentation-driven projects
- Institutional or research sites
- Structured portfolios
- Public-facing informational platforms

It is suitable as a foundation for further customization.

## Accessibility Features

- WCAG AAA color contrast palette
- Visible 4px keyboard focus outlines
- Skip-to-content link
- Reduced-motion support
- Responsive layout (desktop, tablet, mobile)
- Semantic HTML structure
- No JavaScript required

## Design Principles

Inspired by De Stijl:

- Grid-based layout
- Primary color geometry
- Clear hierarchy
- Minimal visual noise

Adapted for modern accessibility and responsiveness.

## Technical Characteristics

- Pure HTML and CSS
- CSS Grid layout
- System font stack
- No external dependencies
- Production-ready structure
